
# Wmall电商网站 （Wmall q前端实现）


### 运行时注意
1. 打开nginx服务器
2. tomcat mysql
3. ftp服务器
3. 内网穿透，natapp
4. npm run dev_win


### 介绍
- 采用前后端分离架构思想
- 利用ajax渲染技术
- 进行调用接口进行页面渲染
- 模块化开发的前端电商项目
- 与服务端真实数据对接。

#### 此项目利用SSM框架，使用了一下技术

1. nginx 负载均衡，图片上传图片服务器
2. mybatis分页插件使用
3. 支付宝真实对接
4. ftp服务器连接
5. 使用mybatis插件进行自动生成dao pojo层
